Is folder mein apni keystore file rakhni hai:
Filename: getredeemcode.jks

AndroidIDE Terminal se keystore banane ka tarika:
  keytool -genkey -v \
    -keystore getredeemcode.jks \
    -alias getredeemcode \
    -keyalg RSA \
    -keysize 2048 \
    -validity 10000

Banane ke baad is file ko yahan (keystore/ folder me) rakh do.
Phir app/build.gradle.kts mein password update karo.

NOTE: Debug APK build ke liye keystore zaroorat NAHI hai.
      Sirf Release APK/AAB ke liye keystore chahiye.
